Lógica de Tiempos y Empalme
RECOJO → AEROPUERTO
H.inicio = menor Hora parada del grupo
H.fin = Hora llegada del planillón
Siguiente servicio empalma desde H.fin
ZARPE ← AEROPUERTO
H.inicio = Hora llegada + espera tipo vuelo
Paradas = Hora parada + espera tipo vuelo
H.fin = ETA calculado al último drop
